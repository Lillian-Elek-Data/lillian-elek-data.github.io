Problem Statement

Stroke is a leading cause of death and disability worldwide, but many risk factors are preventable with early intervention. This project aimed to build a predictive model that could identify high-risk patients based on readily available clinical and demographic data.

Business Impact: Early identification of high-risk patients enables healthcare providers to implement preventive measures, potentially saving lives and reducing healthcare costs associated with stroke treatment and rehabilitation.

Challenges

  • Class Imbalance: Only 4.9% of patients experienced strokes, requiring careful handling of imbalanced data
  • Missing Data: Real-world healthcare data contained missing values that needed strategic imputation
  • Feature Selection: Identifying which clinical markers were truly predictive versus correlated
  • Model Interpretability: Ensuring the model could be explained to healthcare professionals for clinical adoption

Methodology

This project followed a rigorous data science workflow, emphasizing reproducibility and statistical validity.

  • 1. Data Acquisition & Exploration

    Analyzed 5,110 patient records with 12 features including age, BMI, glucose levels, and medical history. Conducted exploratory data analysis to understand distributions, correlations, and data quality issues.

  • 2. Data Cleaning & Preprocessing

    Handled missing values using domain-informed imputation strategies. Removed outliers and addressed data quality issues. Encoded categorical variables (work type, residence, smoking status) and scaled numerical features for model compatibility.

  • 3. Feature Engineering

    Created interaction terms between related health metrics. Engineered age groups and BMI categories based on medical guidelines. Applied polynomial features for non-linear relationships between predictors and stroke risk.

  • 4. Handling Class Imbalance

    Implemented SMOTE (Synthetic Minority Over-sampling Technique) to balance the dataset. Compared performance against undersampling and class weight adjustment approaches to find optimal strategy.

  • 5. Model Development & Training

    Built logistic regression classifier with regularization (L2). Used 80/20 train-test split with stratification to preserve class distribution. Applied cross-validation (5-fold) to ensure model stability and prevent overfitting.

  • 6. Model Evaluation & Validation

    Evaluated using accuracy, precision, recall, F1-score, and ROC-AUC. Analyzed confusion matrix to understand false positive/negative trade-offs. Examined feature coefficients for interpretability and clinical relevance.

  • 7. Statistical Diagnostics

    Performed hypothesis testing on coefficient significance. Checked for multicollinearity using VIF (Variance Inflation Factor). Validated model assumptions including linearity of log-odds and independence of observations.

Key Findings & Insights

Top Predictive Features

The model identified several strong predictors of stroke risk:

  • Hypertension: Patients with hypertension had 2.8x higher stroke risk
  • Heart Disease: Pre-existing heart conditions increased odds by 2.2x
  • Average Glucose Level: Elevated glucose (>126 mg/dL) strongly correlated with risk
  • BMI: Obesity (BMI > 30) contributed to increased risk, though effect was smaller than expected

Model Performance Analysis

Precision vs. Recall Trade-off: The model achieved 89% precision and 87% recall. In a clinical context, this means:

  • High Precision: When the model predicts stroke risk, it's correct 89% of the time (low false alarms)
  • High Recall: The model catches 87% of actual high-risk patients (minimal missed cases)
