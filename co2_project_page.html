Emissions by the Equation: Predicting CO₂ with OLS

A regression-powered deep dive into what drives state-level carbon emissions across the United States.

This project uses Ordinary Least Squares regression to predict CO₂ emissions based on environmental and economic factors. Through log transformations, feature selection, and comprehensive diagnostics, I built a high-accuracy model revealing the key drivers of carbon emissions at the state level.

View on GitHub Back to Projects

Project Overview

Understanding the factors that drive carbon emissions is crucial for developing effective environmental policies. This project analyzes 2009 state-level data to identify and quantify the relationship between various environmental, economic, and demographic factors and CO₂ emissions.

Objective

Build a predictive OLS regression model to understand which factors most significantly impact state-level CO₂ emissions

Dataset

2009 state-level environmental and economic data including emissions, population, energy consumption, and economic indicators

Outcome

High-accuracy predictive model with comprehensive residual diagnostics and clear insights into emission drivers

Methodology

This project follows a rigorous machine learning workflow with emphasis on statistical validity and model interpretability.

Data Preparation & Exploration

  • Loaded and cleaned 2009 state-level emissions dataset
  • Performed exploratory data analysis (EDA) to understand distributions and relationships
  • Identified potential predictors of CO₂ emissions
  • Checked for missing values, outliers, and data quality issues

Feature Engineering & Transformation

  • Applied log transformations to handle skewed distributions
  • Evaluated correlation between potential predictors
  • Selected optimal features through statistical analysis
  • Addressed multicollinearity concerns

Model Development

  • Implemented Ordinary Least Squares (OLS) regression using statsmodels
  • Split data into training and testing sets for validation
  • Trained multiple model configurations to find optimal predictors
  • Validated OLS assumptions (linearity, normality, homoscedasticity, independence)

Evaluation & Diagnostics

  • Conducted comprehensive residual analysis
  • Evaluated model performance using R², adjusted R², and RMSE
  • Created diagnostic plots (residual plots, Q-Q plots, scale-location)
  • Assessed prediction accuracy on hold-out test set
  • Interpreted coefficient significance and effect sizes

Key Findings

Model Performance

  • Achieved high R² indicating strong predictive power
  • Low RMSE demonstrating accurate predictions
  • Residuals showed normal distribution with constant variance
  • Model successfully validated on unseen test data

Significant Predictors

  • Economic activity strongly correlates with emissions
  • Population density influences emission patterns
  • Energy consumption is a primary driver
  • Log transformations revealed nonlinear relationships

Business Impact

This analysis provides actionable insights for environmental policymakers and researchers. By understanding which factors most significantly impact CO₂ emissions, states can develop targeted interventions to reduce their carbon footprint. The model's high accuracy enables reliable forecasting for policy scenario planning.

Implementation Highlights

Train-Test Split Validation

Implemented proper data splitting to ensure model generalizability. The model was trained on a subset of states and validated on held-out data to prevent overfitting and ensure reliable performance estimates.

Log-Transformed Modeling

Applied logarithmic transformations to handle skewed distributions and capture nonlinear relationships. This transformation improved model fit and made interpretation more meaningful for multiplicative effects.

Comprehensive Diagnostics

Conducted full residual diagnostics including Q-Q plots, residual vs. fitted plots, and scale-location plots to validate OLS assumptions. Ensured linearity, normality, homoscedasticity, and independence of residuals.

Statistical Significance Testing

Evaluated p-values and confidence intervals for all predictors to determine statistical significance. Interpreted coefficients to understand the magnitude and direction of each variable's effect on emissions.
